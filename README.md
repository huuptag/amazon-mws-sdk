# Amazon MWS SDK

##### Install SDK by composer:

<code>composer require huule\\amazon-sdk dev-master</code>

##### Or setup local package:

- Create composer.json file:

```json
{
    "require": {
        "huule/amazon-sdk": "dev-master"
    },
    "repositories": [
        {
            "type": "path",
            "url": "../amazon_sdk/"
        }
    ]
}
```

- Run command ```composer install```
